# Personaje2024
## Nombre del personaje
Árbol de Navidad
## Creador
Cruz Méndez Juan Gustavo Ángel
## Explicacion de funcionamiento
El árbol navideño tendrá la funcionalidad de encender luces led, reporducir un villancicos, ademas de retirarse y colocarse un gorrito navideño cuando una 
persona se acerque.
